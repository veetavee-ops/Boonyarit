## 2. Bug Fix — Bot ไม่อัปโหลดรูปภาพขึ้น Google Drive

**Root cause:** `handleImageMessage()` ไม่ได้ส่ง `folderName` ต่อไปให้ `saveImageGroup()` ทำให้รูป group ไม่ถูก upload ขึ้น Drive

**แก้ใน `backend/routes/webhook.js`:**
- ส่ง `folderName` เข้า `handleImageMessage()`
- capture `senderName` และเก็บ `folderName` + `senderName` ไว้ใน `pendingImageGroups`
- เพิ่ม Drive upload loop ใน `saveImageGroup()` เหมือน pattern ของ file upload

**Deploy:** push ขึ้น GitHub → GitHub Actions build Docker image → deploy production อัตโนมัติ
